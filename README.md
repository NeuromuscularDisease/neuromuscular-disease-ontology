# Neuromuscular Disease Ontology

Description: The Neuromuscular Disease Ontology (NMDO) is a project ontology built to support the mapping and integration of NMD datasets.

## Versions

### Stable release versions

The latest version of the ontology can always be found at:

[https://github.com/NeuromuscularDisease/neuromuscular-disease-ontology/releases](https://github.com/NeuromuscularDisease/neuromuscular-disease-ontology/releases)

### Editors' version

Editors of this ontology should use the edit version, [src/ontology/nmdo-edit.owl](src/ontology/nmdo-edit.owl)
